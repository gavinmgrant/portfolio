Projects

Happy Hour Helper

This app recommends food and cocktail recipes for users planning a happy hour. Users first select if they want to search for food or cocktail recipes, then are taken to an options view. After search queries and options are selected, the user clicks the go button to see their results. At the end detail results are provided as well as the option to start over.

See GitHub repo

Technologies used:

  • HTML5
  • CSS
  • JavaScript
  • jQuery
  • RESTful APIs

Name This City

This is an urban geography quiz app that tests your knowledge of cities across the globe. The user clicks through ten questions while a score gets updated and has the option to start over. Each question is accompanied by a photo of the city to help the user determine the right answer. At the end of the quiz, the total score is displayed along with a message of how well the user did.

See GitHub repo

Technologies used:

  • HTML5
  • CSS
  • JavaScript
  • jQuery
