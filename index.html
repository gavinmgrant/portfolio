Permitful

This full-stack web application allows users to visualize where all of the most recently updated building permits in a specific jurisdiction are located and save their favorites.

Built a responsive client using JavaScript and React.js framework that integrates with the Google Maps and Socrata Open Data APIs.

Architected RESTful API endpoints using Node.js/Express routers to allow users to save favorite permits and to implement an authentification system with JWT.

Engineered a PostgreSQL database to save user and favorite permit data while seemlessly integrating with the Express application.

Implemented client- and server-side testing using Enzyme, Supertest, Mocha, and Chai.
